This filter includes/excludes jobs that haven't already been included in another view.

Rationale

Useful if you have hundreds of jobs organized into views, and you want an easy way to identify which jobs haven't been "classified" into a view yet.

Options

Match type:
Whether to add or remove jobs from this view. There's two kinds of match types:
  • Include matched/unmatched: adds additional jobs to this view.
  • Exclude matched/unmatched: removes jobs from this view that have been included by previous filters.

Note

Because this filter looks at other views, there is a risk of creating circular references between views which can lead to unpredictable behavior. This filter will warn you if that's the case.
