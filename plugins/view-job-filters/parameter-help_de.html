Dieser Filter inkludiert/exkludiert Jobs anhand ihrer Parameter.

Optionen

Name:
Regulärer Ausdruck der auf den Namen eines Parameters zutreffen muss, damit dieser Filter auf den Job zutrifft. Siehe die vollständige Liste der verfügbaren Ausdrücke. Wenn dieses Feld leer ist, trifft jeder Parameter-Name zu.
Wert:
Regulärer Ausdruck der auf den Wert eines Parameters zutreffen muss, damit dieser Filter auf den Job zutrifft. Siehe die vollständige Liste der verfügbaren Ausdrücke. Wenn dieses Feld leer ist, trifft jeder Parameter-Wert zu.
Beschreibug:
Regulärer Ausdruck der auf die Beschreibung eines Parameters zutreffen muss, damit dieser Filter auf den Job zutrifft. Siehe die vollständige Liste der verfügbaren Ausdrücke. Wenn dieses Feld leer ist, trifft jede Parameter-Beschreibung zu.
Filter-Typ:
Gibt an ob Jobs hinzugefügt oder entfernt werden sollen. Es gibt zwei mögliche Arten von Filter-Typen:
  • Zutreffende/Unzutreffende aufnehmen: Fügt zusätzliche Job zu dieser Ansicht hinzu.
  • Zutreffende/Unzutreffende auschließen: Entfernt Jobs aus dieser Ansicht, die durch vorherige Filter hinzugefügt wurden.

Anmerkungen

Dieser Filter funktioniert für folgende Parameter-Typen: Text, Boolscher Wert, Datei, Auswahl and validierender Text.
