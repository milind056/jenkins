Dieser Filter inkludiert/exkludiert Jobs abhängig davon ob die ID oder der Name des aktuellen Benutzers irgendwo vorkommt.

Optionen

Namenssuche:
Gibt an, ob die Jobs nach der ID oder dem Namen des aktuellen Benutzers durchsucht werden sollen. Der Namensvergleich kann dahingehend angepasst werden, dass bestimmte Buchstaben ignoriert werden.
Zutreffende Jobs:
Gibt an, welche Eigenschaften eines Jobs nach der ID oder dem Namen des aktuellen Benutzers durchsucht werden.
Zutreffende Builds:
Gibt an, wie viele Builds die ID oder den Namen des aktuellen Benutzers enthalten müssen, damit dieser Filter auf den Job zutrifft.
Zurückliegender Bereich:
Ein Job kann viele Builds haben und diese Option bestimmt, wie viele dieser Builds zur Überprüfung herangezogen werden. Die Angabe geht rückwärts vom letzten Build aus und kann eine Dezimalzahl sein, z.B. 1.5 Stunden. Um alle Builds heranzuziehen kann 0 als Wert angegeben werden.
Filter-Typ:
Gibt an ob Jobs hinzugefügt oder entfernt werden sollen. Es gibt zwei mögliche Arten von Filter-Typen:
  • Zutreffende/Unzutreffende aufnehmen: Fügt zusätzliche Job zu dieser Ansicht hinzu.
  • Zutreffende/Unzutreffende auschließen: Entfernt Jobs aus dieser Ansicht, die durch vorherige Filter hinzugefügt wurden.
