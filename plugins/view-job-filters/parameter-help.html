This filter includes/excludes jobs based on their parameters.

Options

Name:
Regular expression that needs to match the parameter name in order for this filter to match the job. See the full list of available patterns. Leave this field blank if you want to match any name.
Value:
Regular expression that needs to match the parameter value in order for this filter to match the job. See the full list of available patterns. Leave this field blank if you want to match any value.
Description:
Regular expression that needs to match the parameter description in order for this filter to match the job. See the full list of available patterns. Leave this field blank if you want to match any description.
Match type:
Whether to add or remove jobs from this view. There's two kinds of match types:
  • Include matched/unmatched: adds additional jobs to this view.
  • Exclude matched/unmatched: removes jobs from this view that have been included by previous filters.

Note

This filter works with the following parameter types: String, Boolean, File, Choice, and Validating String.
