This filter includes/excludes jobs based on their type.

Options

Job Type:
Which type of job this filter should match. If the dropdown doesn't have the type of job you're looking for that means you don't have the necessary plugin installed.
Match type:
Whether to add or remove jobs from this view. There's two kinds of match types:
  • Include matched/unmatched: adds additional jobs to this view.
  • Exclude matched/unmatched: removes jobs from this view that have been included by previous filters.
