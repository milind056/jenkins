Dieser Filter inkludiert/exkludiert Jobs anhand ihres SCM-Typs.

Nutzen

Nützlich vor allem in Organisationen mit hunderten von Jobs oder wenn man dabei ist Jobs auf ein neues SCM umzustellen.

Options

SCM-Typ:
Gibt an auf welche SCM-Typen dieser Filter zutreffen soll. Wenn ein Typ von SCM nicht in der Liste auftaucht, dann liegt das daran, dass das entsprechende Plugin nicht installiert ist.
Filter-Typ:
Gibt an ob Jobs hinzugefügt oder entfernt werden sollen. Es gibt zwei mögliche Arten von Filter-Typen:
  • Zutreffende/Unzutreffende aufnehmen: Fügt zusätzliche Job zu dieser Ansicht hinzu.
  • Zutreffende/Unzutreffende auschließen: Entfernt Jobs aus dieser Ansicht, die durch vorherige Filter hinzugefügt wurden.
